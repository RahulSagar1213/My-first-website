Object Oriented Programming:

Object Oriented Programming is a programming language based on the concept of object and class in which data in form of field often known as attributes and code in form of procedure often known as methods. Object Oriented Programming is based on buttom-up approach and program also provide access specifiers like private, public and protected. So, data hiding is possible in object oriented programming and data are secure. It also allows overloading and based on the real world problems. The basic concept of Object Oriented Programming are inheritance, data abstraction, encapsulation and polymorphism.

Class:

Class is an user-defined datatype that allows us to group activites and action together. In other words, class is a collection of objects of similar type which are also userdefined datatype and behaves like built in type in programming language. Once class has been defined, we can create any number of objects associated with that class. 

		Syntax:
          Class class_name {
                                    Acess specifier:
                                            Data member and member function;
